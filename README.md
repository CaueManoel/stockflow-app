# StockFlow App

Aplicativo mobile para **gerenciamento de pedidos, cadastro de produtos e impressão de notas**, integrado ao sistema **StockFlow**.

O app foi desenvolvido para que o **proprietário do negócio possa acompanhar e gerenciar os pedidos em tempo real**, diretamente pelo celular.

---

## 📱 Sobre o Aplicativo

O **StockFlow App** permite que o administrador visualize pedidos realizados no sistema, gerencie produtos cadastrados e imprima rapidamente a nota de cada pedido utilizando uma impressora compatível.

A aplicação foi desenvolvida com foco em **agilidade no atendimento e controle de vendas**.

## 🚀 Como Executar o Projeto

1. Clone o repositório:

```
git clone https://github.com/CaueManoel/stockflow-app.git
```

2. Instale as dependências:

```
npm install
```

3. Inicie o projeto:

```
npm start
```

4. Execute no emulador ou dispositivo físico utilizando **Expo** ou **React Native CLI**.
